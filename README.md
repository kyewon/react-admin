## Description

react, redux, hook, saga, typescript, styled-components 등을 사용한 boilerplate 입니다.

동작 테스트: [Demo](https://mxxnlighter.s3.ap-northeast-2.amazonaws.com/uploads/admin/build/index.html)


## Installation

```bash
npm install
```

## Running

```bash
npm run start
```

## Build

```bash
npm run build
```

## Testing

```bash
npm run test
```
